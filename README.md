# LETRAX — Infinite Wordle

Wordle infinito com palavras em inglês. Sem limite de partidas por dia.

## Features
- 🎮 Jogadas ilimitadas
- 🇺🇸 Palavras em inglês (5 letras)
- 📊 Estatísticas persistentes (localStorage)
- 📱 PWA — instale na tela inicial
- ⌨️ Teclado virtual + físico
- 🎨 Tema dark com cores neon

## Deploy na Vercel

1. Suba este repo no GitHub
2. Importe na [Vercel](https://vercel.com/new)
3. Deploy automático — sem configuração extra

Ou via CLI:

```bash
npm i -g vercel
vercel
```

## Stack
HTML + CSS + JavaScript vanilla. Zero dependências.
